This project is a Tic-Tac-Toe game with an AI opponent developed using Python and the Minimax algorithm.
The AI makes optimal decisions, making it unbeatable when played correctly.

This project was completed as part of Week 1 of the WeIntern Internship to understand the fundamentals of Artificial
Intelligence, game logic, and decision-making algorithms.

--What is Minimax?
